
🔍 This project focuses on converting a messy, unclean dataset into meaningful insights using data cleaning, visualization, and statistical analysis.

 Project Includes:
• Data Cleaning & Preprocessing (handling missing values, duplicates, returns)
• Data Analysis using Pandas & NumPy
• Time Series Analysis (daily & monthly sales trends)
• 10+ Visualizations (Line plot, Bar chart, Histogram, Boxplot, Pie chart, etc.)
• Exploratory Data Analysis (EDA)
• Statistical Analysis (Normal Distribution, Regression, Outlier Detection)

 Key Insights:
• Sales data is highly right-skewed (not normally distributed)
• Identified peak sales periods using time series trends
• Top products contribute a major share of total revenue
• Outliers significantly affect overall analysis

🚀 This project helped me understand the complete Data Science workflow — from raw data cleaning to visualization and statistical interpretation using NumPy, Pandas, Matplotlib, and SciPy.
